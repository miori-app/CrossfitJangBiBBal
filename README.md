## 👩🏻‍💻 아키텍쳐 

### MVVM

> **MVVM**
- View 는 Snapkit을 활용해, 화면을 그리는 역할에만 집중하게끔 진행했습니다.
- Model의 데이터 관리, 로직 실행은 ViewModel 에서 진행되도로 했습니다.
